Chat Widget Demo

Features

  • Conversation History: The widget shows conversation history for returning visitors.
  • Automatic New Chat: For first-time visitors, it automatically starts a new conversation.
  • Persistent Sessions: Conversations are stored and can be continued across visits.
  • Embedded Experience: The entire chat experience happens within the widget on your website.
  • Responsive Design: Works well on both desktop and mobile devices.

How to Embed the Chat Widget

Add the following code to your website to embed the chat widget:

<!-- Chat Widget -->
<link rel="stylesheet" href="http://localhost:8080/widget.css">
<script src="http://localhost:8080/widget.js"></script>
<script>
  document.addEventListener('DOMContentLoaded', function() {
    initChatWidget('680cef107640794a082542b1');
  });
</script>

Replace '680cef107640794a082542b1' with your actual chatbot ID.

Customization Options

You can customize the appearance of the widget by adding your own CSS:

<style>
  /* Change button color */
  .chat-widget-button {
    background-color: #2563eb;
  }

  /* Change panel width */
  .chat-widget-panel {
    width: 400px;
  }

  /* Change header color */
  .chat-widget-header {
    background-color: #2563eb;
  }

  /* Change user message color */
  .chat-message-user {
    background-color: #2563eb;
  }
</style>

How It Works

The chat widget uses a visitor ID stored in localStorage to identify returning users. When a visitor clicks the chat button:

  1. If they have previous conversations, the widget shows their conversation history.
  2. If they have no previous conversations, it automatically starts a new chat.
  3. All conversations are stored in the database and associated with the visitor ID.
  4. The visitor can switch between past conversations or start new ones.
